목차

KWTP의 페이지들을 리스트로 나타내었고, 페이지명.html을 누르면 각 페이지로 이동합니다.
그리고 페이지명을 누르면 각 장의 어떤 기술이 사용되었는지 내용이 펼쳐집니다.

  • 메인페이지( index.html)
    •
    2장의 <!DOCTYPE html>, <html>, <head>, <title>, <body>, <p>, <br>, <strong>, <div>, <link>, <style>, <script>, <meta>, <img>, <ul>, <li>, <a> 등의 태그 사용
    3장의 <header>, <nav>, <section>, <footer>, <details>, <summary>, <form>, <input>, <button> 등의 태그 사용
    4장의 <style> 태그로 스타일시트작성, style 속성에 스타일 시트 작성, 셀렉터, 색 표현 속성, 텍스트 꾸미기 속성, 폰트 제어 속성, 박스모델 속성 등 사용
    5장의 배운 블록박스와 인라인박스, 유동 배치, 리스트 꾸미기 속성 등 사용
    6장의 배운 <script> 태그로 자바스크립트 코드 작성, HTML 태그의 이벤트 리스너 속성에 코드 작성, 다이얼로그, 변수, switch문, break문, 함수 등 사용
    8장의 DOM객체, 9장의 배운 onclick 리스너
    location 객체 등 사용
